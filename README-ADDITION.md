
## Надёжность резервных копий

```bash
sudo vpn backup
vpn backup list
vpn backup verify latest
```

Восстановление сначала показывает план и ничего не меняет:

```bash
sudo vpn restore latest
```

Для подтверждённого восстановления:

```bash
sudo vpn restore latest --confirm
```

Перед восстановлением автоматически создаётся аварийная копия текущего состояния.
Если конфигурация Xray или проверка порта не проходит, выполняется автоматический откат.
