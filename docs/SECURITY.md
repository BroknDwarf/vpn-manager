# Безопасность

- Не сохраняйте UUID, privateKey Reality, токены Telegram и пароли в Git.
- Файл `/etc/vpn-manager.conf` устанавливается с правами `600`.
- Диагностика не отправляет данные наружу.
- Перед обновлением 3x-ui/Xray создавайте backup.
- Не используйте `curl | bash` для непроверенных веток или форков.
